# 说明
`dcos-cluster list` 命令允许您列出连接到 DC/OS CLI 的群集。

# 使用

```bash
dcos cluster list [--attached --json]
```

# 选项

| 名称、简写 | D 描述 |
|---------|-------------|
| | `--attached` | 仅附加的群集。 |
| | `--json` | 显示 JSON 格式化列表。 |
