# 说明
`dcos-cluster remove` 命令允许您从 DC/OS CLI 中删除连接的群集。

# 使用

```bash
dcos cluster remove [<cluster-name> | <cluster-id> | --all]
```

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<cluster-name>` | 已连接的群集的名称。 |
| `<cluster-id>` | 已连接的群集的 ID。 |
