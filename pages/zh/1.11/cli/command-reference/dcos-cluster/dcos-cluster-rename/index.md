# 说明
`dcos cluster rename` 命令允许您重命名群集。

# 使用

```bash
dcos cluster rename <name> <new-name>
```

# 位置自变量

| 名称，简写 | 说明 |
|---------|-------------|
| `<name>` | 已连接的群集的名称。 |
| `<new-name>` | 连接的群集的新名称。|
